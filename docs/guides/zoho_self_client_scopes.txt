Zoho Self Client - スコープ設定

=== Scope 欄にコピー&ペースト ===

ZohoProjects.projects.READ,ZohoProjects.tasks.ALL,ZohoProjects.files.READ,ZohoWorkDrive.files.ALL,ZohoWorkDrive.files.READ

=== Scope Description 欄 (オプション) ===

MCP Server for Zoho Projects and WorkDrive integration. Allows reading project information, managing tasks, and accessing files.

=== Time Duration ===

現在: 3 minutes
推奨: 10 minutes (認証作業用)

=== 注意事項 ===
- スコープはカンマ区切りで入力
- スペースは入れない
- 正確にコピー&ペーストしてください
